Open Source

Neptune is an open source library licensed under the Apache License 2.0. It is free to use for commercial purposes and can be modified to custom needs. It operates with the sole purpose of making development more accesible. The source code is available on github. Everything used to develop Neptune is included in the repository. It can be compiled from scratch or a precompiled binary can be downloaded.

Github

Extensible

Neptune uses a system of automatically implemented callbacks. These allow for the extension of input, error, and frame management systems. Rendering exists on a loop that can be either setup by a user or based on a system timer. This allows neptune to be used for both small hobbyist games or large, commerical products.
