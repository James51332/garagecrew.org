NeptuneGL

NeptuneGL is a powerful abstraction library that is build in C99. It provides the essential foundation for Neptune. Not only does it handle window and context creation, but it manages all of Neptune's native api tasks. It is open source and licensed under the Apache 2.0 License.

Extensible

Apollo is an entirely open source project, meaning that anyone can make improvements or changes that can be accepted by the community. Because Apollo is written in C++17, developers around the world can make it more accesible or powerful.

No Dependency Mess

Apollo uses very minimal dependencies, and will reduce that number as time passes. All will be properly setup in a simple recursive git clone. The only tool required to compile neptune is a simple compiler toolchain. The build system allows for compilation using Xcode, Microsoft Visual Studio, or even just GNU Make.
